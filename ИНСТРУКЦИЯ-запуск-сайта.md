# DECOROOMS — Запуск сайта и админ-панели

Это пошаговая инструкция. Делайте по порядку, ничего не пропуская. Всё бесплатно, банковская карта не нужна.

В конце у вас будет:
- Живой сайт в интернете
- Админ-панель по адресу `вашсайт/admin`, где вы добавляете проекты через форму, без кода

---

## Что вам понадобится

- Файлы сайта (папка `decorooms-site` из архива)
- Электронная почта
- 20–30 минут времени

---

## ШАГ 1. Создать аккаунт GitHub

GitHub — это бесплатное хранилище для файлов вашего сайта.

1. Откройте https://github.com/signup
2. Введите почту, придумайте пароль и имя пользователя (например `decorooms`)
3. Подтвердите почту по письму, которое придёт от GitHub
4. Когда спросят про план — выберите бесплатный (**Free**)

Готово, аккаунт есть.

---

## ШАГ 2. Загрузить сайт на GitHub

1. Зайдя в GitHub, нажмите зелёную кнопку **New** (или плюс справа сверху → **New repository**)
2. В поле **Repository name** впишите: `decorooms-site`
3. Выберите **Public**
4. Нажмите **Create repository**
5. На открывшейся странице найдите ссылку **uploading an existing file** (или кнопка **Add file → Upload files**)
6. **Распакуйте архив** на компьютере. Откройте папку `decorooms-site` и выделите всё, что внутри неё (файлы и папки), — перетащите в окно загрузки GitHub

   ВАЖНО: загружать нужно **содержимое** папки `decorooms-site`, а не саму папку. То есть `index.html` должен оказаться в корне репозитория.

7. Внизу нажмите зелёную кнопку **Commit changes**

Дождитесь окончания загрузки (фото много, может занять несколько минут).

---

## ШАГ 3. Подключить хостинг Netlify

Netlify возьмёт файлы из GitHub и сделает сайт доступным в интернете.

1. Откройте https://app.netlify.com/signup
2. Нажмите **Sign up with GitHub** (вход через ваш GitHub-аккаунт — это удобнее всего)
3. Разрешите доступ, когда GitHub спросит
4. На главной Netlify нажмите **Add new site → Import an existing project**
5. Выберите **Deploy with GitHub**
6. В списке найдите ваш репозиторий `decorooms-site` и нажмите на него
7. Настройки оставьте по умолчанию, нажмите **Deploy**

Через минуту сайт будет живой. Netlify даст адрес вида `random-name-123.netlify.app` — это уже работающий сайт. Адрес можно поменять в настройках (**Site configuration → Change site name**).

---

## ШАГ 4. Включить вход в админку (Netlify Identity)

Чтобы админка пускала только вас, нужно включить авторизацию.

1. В панели Netlify откройте ваш сайт
2. Зайдите в **Integrations** (или **Site configuration**) и найдите **Identity** → нажмите **Enable Identity**

   *(В новых версиях Netlify это может называться по-другому или предлагать перейти на «Netlify Identity» как дополнение — следуйте подсказкам на экране, функция бесплатна для небольшого числа пользователей.)*

3. В разделе **Identity → Registration** выберите **Invite only** (чтобы посторонние не регистрировались)
4. В разделе **Identity → Services → Git Gateway** нажмите **Enable Git Gateway**

---

## ШАГ 5. Пригласить себя в админку

1. В разделе **Identity** нажмите **Invite users**
2. Введите свою почту, отправьте приглашение
3. На почту придёт письмо — откройте его и нажмите **Accept the invite**
4. Вас перекинет на сайт, попросит **придумать пароль** — задайте его

Теперь у вас есть логин и пароль для админки.

---

## ШАГ 6. Зайти в админку и добавить проект

1. Откройте `вашсайт/admin` (например `decorooms.netlify.app/admin`)
2. Войдите по своей почте и паролю
3. Вы увидите раздел **«Проекты портфолио»**
4. Нажмите **New Проект** (New Project)
5. Заполните форму:
   - **Название проекта** — например `Parkwood Residence`
   - **Обложка** — загрузите главное фото проекта
   - **Стиль** — например `Современная классика`
   - **Площадь** — число, например `120`
   - **ЖК / Локация** — необязательно
   - **Порядок показа** — меньше число = выше в списке
   - **Галерея фото** — необязательно, дополнительные фото
   - **Описание** — необязательно
6. Нажмите **Publish → Publish now**

Через минуту проект появится на главной странице сайта в портфолио. Готово!

---

## Как это работает дальше

- Каждый раз, когда вы добавляете/меняете проект в админке и нажимаете Publish — сайт сам обновляется через 1–2 минуты.
- Менять код руками больше не нужно.
- Фото проектов загружаются прямо через форму.

---

## Если что-то пошло не так

- **Сайт не открывается после деплоя** — подождите 2–3 минуты, обновите страницу.
- **Админка не пускает** — проверьте, что прошли ШАГ 4 и ШАГ 5 полностью (Identity + Git Gateway включены, приглашение принято).
- **Проект не появился** — откройте сайт в режиме инкогнито или обновите страницу с очисткой кэша (Ctrl+Shift+R).
- **Фото грузятся медленно** — это нормально при первой загрузке большого числа изображений.

## Важное замечание про существующие 4 проекта

Ваши текущие проекты (Constellation, Bois & Lumière, Lumière Naturelle, Résidence Dorée) уже отображаются на главной — они прописаны в файле `projects.json`. Их отдельные красивые страницы с галереями (`projects/*.html`) остаются как есть.

Новые проекты, добавленные через админку, будут показываться карточкой на главной. Если для нового проекта нужна такая же большая страница-галерея, как у Constellation, — это отдельная задача, напишите мне, и мы её настроим.
